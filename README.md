# Machine Learning Aplicado con Python

## Instrucciones de configuración

- Usuarios de Windows, puedes seguir las instrucciones en [SETUP-Windows.md](SETUP-Windows.md)
- Usuarios de Mac, puedes seguir las instrucciones en [SETUP-mac.md](SETUP-mac.md)
- Usuarios de Linux, ¡las instrucciones para Mac deberían ser lo suficientemente precisas!


### Activacion del entorno y Jupyter Lab
1. En Windows abriremos Anaconda Prompt y en macOS una terminal 
2. Debemnos situarnos en la raiz de nuestro proyecto `cd C:\Dev\Machine Learning Aplicado con Python` y ejecutamos esta linea para entrar en el entorno virtual: 
```
conda activate ml
```

3. Ahora arrancamos Jupyter Lab
```
jupyter lab
```
